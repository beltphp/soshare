> URL Shares

Belt.`Soshare` is an utility library that allows you to easily check the number
of shares an URL has for a given social network (or all social networks).

Supported networks:

 - Twitter
 - Facebook
 - LinkedIn
 - Google+
 - reddit
 - Pinterest
 - StumbleUpon

## Installation

Via Composer.

```shell
$ composer require belt/soshare
```
